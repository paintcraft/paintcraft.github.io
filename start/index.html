Start with the Paintcraft

Welcome to Paintcraft! Thanks so much for the interest in this package – we think you're really going to like using Paintcraft to build and launch your high-quality drawing applications.

We are doing our best to make this pacakage artist firendly and really flexible so it will satisfy any of your needs. This quick tutorial is intended to walk anyone familiar with Unity3D through the basics of Paintcraft package in just about 10 minutes. Hopefully you'll learn how to make drawing ap as fast as possible.
